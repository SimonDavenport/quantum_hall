51 //! \brief The main function performs a number of tasks:
52 //! - It calls a function to parse the command line and set the various
53 //! parameters stored in global data structs.
54 //! - It initializes a Monte Carlo configuration for the chosen
55 //! geometry and wave function type
56 //! - It performs thermalization steps on the Monte Carlo configuration
57 //! using the Metropolis method.
58 //! - It gathers sample configurations and writes these to files, optionally
59 //! along with a file containing the wave function values.
60 //!
61 //! With the Metropolis method employed here, sample configurations are taken from
62 //! the probability distribution |psi|^2 associated with a given ground state fractional
63 //! quantum Hall wave function.
