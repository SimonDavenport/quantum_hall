54 //! \brief The Term class contains data to represent a single term in the
55 //! entanglement Hamiltonian, or a bra or ket vector in its Hilbert space.
56 //!
57 //! All terms are built from U(1) current operators in a Virasoro algebra.
58 //!
59 //! If Term is constructed with m_deltaNa0 assigned, then it represents a
60 //! bra or ket vector with reference to a particular particle cut. Alternatively,
61 //! it can be constructed without m_deltaNa0 assigned, in which case it
62 //! represents an operator.
63 //!
64 //! Combinations of such objects will be combined in the correct way such
65 //! that e.g. operators acting on bras/ke4.08248e-01 K_-1 K_-1 K_-1 |-1 1 >
66 //!
67 //! Example:
68 //! The operator 4 J_1 K_1 K_-2 J_-3 with N_A=4 would be represented as:
69 //! - m_coefficient = 4.0
70 //! - m_opArray = {1,1,-2,-3}
71 //! - m_opType = {J,K,K,J}
72 //! - m_deltaNa0 = {4}
73 //!
