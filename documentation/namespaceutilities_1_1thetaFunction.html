The generalised Jacobi theta function is given by .

The following a and b parameters in the generalized theta function correspond to the standard definitions of Theta_1,...,Theta_4:

//!                     a        b  
//!     Theta_1        0.5      -0.5
//!     Theta_2        0.5      0.0 
//!     Theta_3        0.0      0.0 
//!     Theta_4        0.0      0.5  
//! 
//! 
 Note: the generalized theta functions are quite badly behaved.
 So it's important to check the convergence of the summations carefully

\return   complex LOG of theta function
Parameters
aFirst parameter in generalized theta function
bSecond parameter in generalized theta function
zComplex argument used as written in the description
tThe 'nome' of the theta function, used as written in the description. N.B. sometimes defined as exp(q)
