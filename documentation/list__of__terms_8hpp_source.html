40 //! \brief The ListOfTerms class contains functions to perform recursive commutation
41 //! operations on matrix elements of Hamiltonians constructed form U(1)
42 //! current operators and Majora operators.
43 //!
44 //! The Hamiltonian is represented by a vector of Term objects. The bra and
45 //! ket forming the matrix element is defined in the constructor. The class
46 //! assumes the following relations are true:
47 //!
48 //! - [J_n,J_m]= delta_{n+m,0} n for all U(1) currents J,K,...
49 //! - [J_n,K_m]=0 for all U(1) currents J,K,...
50 //! - J_{n>0} |0> = 0 and <0| J_{n<0}
51 //! - J_0 |0> = N
52 //! - [Y_n,Y_m] = delta_{n+m,0} for all Majorana operators (cahrged currents)
53 //! = [Y_n,J_m] = 0
54 ////////////////////////////////////////////////////////////////////////////////
