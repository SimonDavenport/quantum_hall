2 //!
3 //! \author Simon C Davenport and Ivan D Rodriguez
4 //!
5 //! \date Last Modified: 03/06/2015
6 //!
7 //! \mainpage
8 //! This suite of programs contains a number of codes used to calculate
9 //! the real space entanglement spectrum (RSES) of model integer and
10 //! fractional quantum Hall wave functions.
11 //!
12 //! INSTALLATION:
13 //!
14 //! To install this code you will first need to download and install
15 //! the following libraries:
16 //!
17 //! - https://gmplib.org/ (high precision library)
18 //! - http://www.mpfr.org/ (high precision library)
19 //! - http://www.multiprecision.org/ (high precision library)
20 //! - http://www.boost.org/ (we only need to the program options)
21 //! - https://www.gnu.org/software/gsl/ (for numerical quadrature)
22 //! - http://fftw.org/ (for fast Fourier transforms)
23 //!
24 //! Once this is completed, you can install our codes using a
25 //! standard GNU build system.
26 //!
27 //! - to see the configuration options run ./configure --help
28 //! - when you are happy run ./configure [options]
29 //! (note that the installation directory is set with --prefix)
30 //! - modify the options/check library installations until
31 //! all the configure tests are passed.
32 //! - run make -j to compile, and make -j install to move
33 //! the compiled codes to the installation directory. Run make
34 //! clean to tidy up the build (and leave the installation
35 //! unchanged). Run make uninstall to remove the installation.
36 //!
37 //! Now that the code is installed, several methods are available
38 //! to you for RSES calculations:
39 //!
40 //! - fqhe entanglement energy method:
41 //! See entanglement_energy_model.cpp. Treats the RSES for
42 //! fractional quanutm Hall states in a non-interacting approximation
43 //! in terms of a set of single-particle entanglement energy levels.
44 //! Author: Simon C Davenport
45 //!
46 //! - fqhe entanglement hamiltonian method:
47 //! See fqhe_entanglement_hamiltonian.cpp. Treats the RSES for integer
48 //! or fractional quantum Hall states in the formalism of Dubail-Read
49 //! and Rezayi's paper Phys. Rev. B 86, 245310 (2012). The formalism
50 //! has also been exteneded in an attempt to treat composite fermion
51 //! fractional quantum Hall states.
52 //! Author: Simon C Davenport
53 //!
54 //! - fqhe entanglement wave function method:
55 //! See entanglement_wave_function.cpp. A specialist method
56 //! based on the "entanglement wave functions" proposed by Rodriguez
57 //! et al. in Phys. Rev. B 88, 155307. This method can be used to
58 //! calculate the RSES of the Laughlin and positive effective field
59 //! Jain states very very large system sizes.
60 //! Author: Ivan D Rodriguez
61 //!
62 //! - fqhe pseudo reduced density matrix method:
63 //! See fqhe_entanglement_spectrum.cpp. A more general method
64 //! to calculate the approximate RSES associated with quantum Hall
65 //! states with known trial wave functions. Based on the
66 //! observations of Rodriguez et al. in Phys. Rev. Lett. 108, 256806.
67 //! Authors: Ivan D Rodriguez and Simon C Davenport
68 //!
69 //! - iqhe exact entanglement spectrum:
70 //! See iqhe_entanglement_spectrum.cpp. Calculates the RSES of the
71 //! integer quanutm Hall states according to the analytical
72 //! result (but doing the requisite integrals numerically).
73 //! Author: Simon C Davenport
74 //!
75 //! There are also several other sets of algorithms and utilities
76 //! bundelled with the code:
77 //!
78 //! - fqhe wave function algorithms:
79 //! See the FQHE namespace. A set of fairly state-of-the-art
80 //! algorithms to evaluate a large number of different types of
81 //! fractional quantum Hall ground state wave function. Mostly
82 //! done in the sphere geometry case, but disk and torus geometry
83 //! is sometimes available.
84 //! Author: Simon C Davenport
85 //!
86 //! - fqhe wave function monte carlo:
87 //! See fqhe_monte_carlo.cpp. An older code containing
88 //! some basic routines to evaluate the ground state energy and
89 //! ground state correlation functions of fractional quantum
90 //! Hall states using monte carlo samlpling.
91 //! Author: Simon C Davenport
92 //!
93 //! - occupation basis:
94 //! See occupation_basis.cpp. Some shared utilities for generating the
95 //! angular momentum basis used by the other RSES calculations
96 //! in the sphere geometry case.
97 //! Author: Simon C Davenport
98 //!
99 //! - program_options:
100 //! See e.g. general_options.h Some shared utilities for program option
101 //! handelling.
102 //! Author: Simon C Davenport
103 //!
104 //! - python codes:
105 //! A set of additional python codes that can be used
106 //! to analyse the RSES data generated by the other programs. In
107 //! particular "fit_entanglement_spectrum.py" can be used to
108 //! fit one set of RSES data to one of the models.
109 //! Author: Simon C Davenport
110 //!
111 //! - utilities:
112 //! See the utilities namespace. A set of general purpose utility
113 //! functions required for low level mathematical functions e.g.
114 //! linear algebra, integer partitions and some C++ hacks.
115 //!
