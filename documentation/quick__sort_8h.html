Detailed Description

Author
Simon C. Davenport
Date
Last Modified: 01/10/2014
This file contains a template algorithm for sorting lists containing

associated pairs of values. The list can be sorted into ascending or descending order. The list can also be partially sorted. Complex numbers are always compared via their absolute values.
