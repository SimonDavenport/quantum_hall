Detailed Description

Author
Simon C Davenport
Date
Last Modified: 19/12/2013

The purpose of this program is to generate statistically random sample configurations of particle co-ordinates, based on the probability distribution |psi|^2 associated with a given ground state fractional quantum Hall wave function.

Start by looking at the main function in fqheMonteCarlo.cpp.

The main function performs a number of tasks:

  • It calls a function to parse the command line and set the various parameters stored in global data structs.
  • It initializes a Monte Carlo configuration for the chosen geometry and wave function type
  • It performs thermalization steps on the Monte Carlo configuration using the Metropolis method.
  • It gathers sample configurations and writes these to files, optionally along with a file containing the wave function values.

With the Metropolis method employed here, sample configurations are taken from the probability distribution |psi|^2 associated with a given ground state fractional quantum Hall wave function.
