Generate the overlap matrix of monopole harmonic functions in the upper half-sphere (assuming an equatorial real space cut) for functions with the same z-component of angular momentum but occurring in different Landau levels.

This calculation is necessary because such functions are not orthonormal when integrated over only half of the sphere. The overlap matrix generated here is diagonalized in order to generate a new set of single particle entanglement energies for construction of the RSES

Parameters
overlapMatrixBufferBuffer to store list of overlap matrices
qMonopole strength
nbrLevelsNumber of Landau levels
nbrStatesNumber of single particle orbitals
